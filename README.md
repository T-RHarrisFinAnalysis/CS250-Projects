# CS-250: Software Development Lifecycle

## Course Description

This repository contains coursework, assignments, journals, and projects completed for **CS-250: Software Development Lifecycle** at Southern New Hampshire University (SNHU).

The course explored the Software Development Lifecycle (SDLC), Agile development methodologies, Scrum practices, software documentation, testing, project planning, and collaboration throughout the software development process.

* Sprint review and retrospective
* Scrum reflections
* Journal reflections

* Supporting course materials

This repository serves as a portfolio of my coursework completed during CS-250 and demonstrates my understanding of modern software development practices and Agile project management.
